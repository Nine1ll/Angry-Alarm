    - 강제 기상 서비스 
        1. 앱 이름
            - AngryAlarm 
        2. 사용 기능
            1. 문자(카카오 API 사용 : 최대 5명//sms 전송)
            2. 시계(알람 설정) 알람음, 
            3. 친구 목록, 메세지 목록   
        3. 세부 기능
            필수기능 (기록자 : 오명빈)
            1. 알람 설정: 사용자가 원하는 시간과 날짜에 알람을 설정 - textview나 edittext 이용
            2. 반복 알람: 사용자가 알람을 특정한 요일이나 주기로 반복하도록 설정
            3. 알람 사운드/진동: 사용자가 알람이 울릴 때 재생할 사운드나 진동 패턴
            4. 알람 해제: 알람이 울리면 사용자는 알람을 해제
            5. 알람 연기: 사용자가 알람을 연기하고 나중에 다시 울릴 시간을 설정
            6. 메세지 전송: 사용자가 알람을 일정 횟수 이상 놓치면 등록한 지인에게 문자 메세지를 전송
            7. 사용자 설정: 사용자가 알람 앱을 맞춤 설정
            8. 알람 기록: 사용자가 알람 기록을 확인
        4. 화면 구성
            (0) 시작 화면
            (1) 알람 설정화면 :

                기본 시간 설정화면 : 날짜 및 반복 일자 설정
                친구 및 메세지 설정화면 -> DB 연동
                알람음 설정화면 : 알람음, 노래(AudioManager 클래스), 진동(Vibrator 클래스) , 다시 울림(간격, 반복) -> DB 연동

            (2)있는 알람 보여주는 화면 -> DB 연동
            (3)화면이 사용 중일 때 알람
            (4)화면이 사용 중이지 않을 때 알람.

        6. 세부 파일 정리 (임시 설계)
        
            XML 파일:

            1. activity_main.xml: 앱의 메인 화면을 담당하는 액티비티의 레이아웃 파일. 이 파일에는 Fragment를 표시할 컨테이너 뷰 (예: FrameLayout, LinearLayout 등)가 포함
            2. fragment_alarm_list.xml: 알람 목록을 표시하는 Fragment의 레이아웃 파일. 알람 목록을 표시하는 RecyclerView나 ListView 등이 포함
            3. fragment_alarm_settings.xml: 알람 설정을 위한 Fragment의 레이아웃 파일. 알람 설정에 필요한 UI 요소들 (시간 선택기, 반복 설정, 알람 사운드/진동 선택 등)이 포함
            4. fragment_alarm_dismiss.xml: 알람 해제 화면을 위한 Fragment의 레이아웃 파일. 알람 해제에 필요한 UI 요소들 (알람 해제 버튼, 연기 버튼 등)이 포함
        
            액티비티:

            1. MainActivity.java: 앱의 메인 화면을 담당하는 액티비티 클래스. 이 액티비티에서는 알람 목록 Fragment와 다른 Fragment들을 관리하고 상호작용을 조정.
        
            Fragment:

            1. AlarmListFragment.java: 알람 목록을 표시하고 관리하는 Fragment 클래스
            2. AlarmSettingsFragment.java: 알람 설정을 위한 Fragment 클래스
            3. AlarmDismissFragment.java: 알람 해제 화면을 담당하는 Fragment 클래스
